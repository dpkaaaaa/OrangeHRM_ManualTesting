# OrangeHRM_ManualTesting

## Project Overview
This project demonstrates manual testing performed on the OrangeHRM demo application.

## Modules Tested
 **Login Module**
 **Admin Module**

## Tools Used
- Microsoft Excel (for Test Cases & Bug Report)
- Google Chrome / Web Browser

## How to Use
1. Download the file `OrangeHRM_Manual_Testing.xlsx` from this repository.  
2. **Open `OrangeHRM_ManualTesting.xlsx` in Microsoft Excel** (or Google Sheets) to view all test cases and bug reports.  
3. Follow each test case to execute testing.  
4. Refer to the Bug Report sheet for identified issues.
